== Description ==

Replace the basic wordpress emoticons with some nice blue emoticons

== Installation ==

1. Upload `blue-smilies/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php smilies_clickable() ?>` in comments.php from your templates
